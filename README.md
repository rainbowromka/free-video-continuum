# Free Video Continuum

Self-hosted система для каталогизации, сравнения и монтажа видео.

## Возможности

- Сканирование локальных дисков и папок с медиафайлами
- Извлечение метаданных через ffprobe
- Ручная каталогизация: теги, описания, субклипы
- Попарное сравнение видео с рейтингом Эло
- Транскодирование прокси-версий (480p, 720p, 1080p)
- Удалённый доступ через промежуточный сервер
- Импорт с флешек, камер, телефонов
- Монтаж с предзагрузкой и бесшовным переключением клипов

## Компоненты

| Компонент | Язык | Описание |
|-----------|------|----------|
| server | Rust | Домашний сервер, REST API, база данных |
| client | Rust | Нативный плеер и каталогизатор |
| shared | Rust | Общие структуры данных |
| relay | Go | Промежуточный сервер (VPS, туннель, авторизация) |
| android-uploader | Android | Слив видео с телефона (позже) |
| web-client | Web | Веб-интерфейс (позже) |

## Технологии

- **Сервер:** Actix-web, SQLite, FFmpeg
- **Клиент:** winit, OpenGL/Vulkan, FFmpeg (библиотека)
- **Туннель:** кастомный TCP/TLS между домашним сервером и VPS
