
## Крейты

### server (Rust)
Домашний сервер. Работает на Windows/Ubuntu.
- Сканирование папок с медиафайлами
- Извлечение метаданных через ffprobe
- Хранение в SQLite
- Транскодирование прокси через ffmpeg
- REST API для клиента
- Туннель к промежуточному серверу

### client (Rust)
Нативный плеер-каталогизатор.
- Воспроизведение видео через FFmpeg (библиотека)
- Отображение через GPU (OpenGL/Vulkan)
- Интерфейс каталогизации и сравнения
- Работа с удалённым сервером через HTTP
- Импорт видео из флешек устройств. Регистрация флешек для удобной каталогизации. Например камера GoPro, Insta, Phone и т.д.

### shared (Rust, библиотека)
Общие структуры данных:
- Asset, Subclip, Tag
- Конфигурация
- DTO для API

### relay (Go)
Промежуточный сервер на VPS.
- Терминирует HTTPS
- Авторизация
- Проброс запросов в туннель к домашнему серверу

### android-uploader (Android, позже)
Приложение-компаньон для слива видео с телефона.
- Находит видео на устройстве
- Отправляет на сервер по Wi-Fi
- Минимальный интерфейс: выбрать файлы, нажать "отправить"

### web-client (позже)
Веб-интерфейс для каталогизации из браузера.

## Технологии

| Компонент | Технология |
|-----------|------------|
| Серверный фреймворк | Actix-web |
| База данных | SQLite (rusqlite) |
| FFmpeg | Внешний процесс (сервер), библиотека (клиент) |
| GUI клиент | winit + OpenGL/Vulkan |
| Туннель | Кастомный TCP/TLS (домашний → VPS) |
| Промежуточный сервер | Go + net/http |
| Android Uploader | Kotlin / Jetpack Compose |
